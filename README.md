# Home Assistant Apps

This is my personal collection of Home Assistant apps. Feel free to browse and use them — they are shared as-is with no guarantees.

## Adding this repository to Home Assistant

1. Go to **Settings → Apps → App Store**
2. Click the menu (⋮) in the top-right and choose **Repositories**
3. Add this repository's URL and click **Add**

## Apps

| App | Description |
|-----|-------------|
| [RTSP to S3](rtsp_to_s3/README.md) | Capture an RTSP camera stream and upload segments to S3 via AWS IoT credentials |
